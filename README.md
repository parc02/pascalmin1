### REQUIREMENTS
- 2명이상의 발표자를 입력받음
- 입력받은 발표자들중 랜덤한 사람을 뽑아서 출력함.

### USE
```
$ npm install -g pascalmin1
$ pascalmin1
hello pascalmin1

$ pascalmin1 발표자1 발표자2 발표자3 (...)
발표자: (Random)발표자
```

### DEV STACK
- WSL > nvm > nodejs v20.11.0 > npm cli


### PUBLISH
```
$ npm login
$ npm publish
```

### RESTRICTION
- 본 프로그램은 Window 환경의 WSL을 사용하여 개발되었음으로 MAC등 기타 환경에서 정상적으로 동작하지 않을 수 있음.
